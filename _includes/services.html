Who we are

We’re a Python user group in Frederick, Maryland for people interested in learning and using the Python programming language.

We exist to teach others how to write code in Python so that they may impact the world around them.

Want to chat? Join us on Slack, our group chat tool. Need help? Check out the Get Help page.

Python Talks

We want people to discover the awesome things Python can do! Python Frederick hosts monthly talks every second Wednesday.

Open Workshops

Every third Saturday we host an open workshop along with the rest of Frederick Tech. Help with a common project, bring your own, or bring your questions!

Book Club

Join us for our book club where we dive into the design of open source software applications, learn the fundamentals of Python, and more!
